# Agentic AI based AUtonomous Cyber Security System
Agent AI monitors network traffic and automatically blocks the IP address of the cyber threats. This was the sample project using CICIDS dataset.

**Overview:**

The Agentic AI Based Autonomous Cyber Security System is an intelligent security solution designed to automatically monitor, detect, and respond to network threats in real time. The system uses machine learning algorithms and AI-based decision-making agents to analyze network traffic and identify malicious activities.

This project aims to reduce the dependency on manual security monitoring by creating an autonomous cybersecurity system capable of detecting intrusions and alerting administrators instantly.


**Problem Statement:**

With the rapid growth of digital infrastructure, cyber threats such as malware attacks, network intrusions, and unauthorized access have become increasingly common. Traditional security systems require constant manual monitoring and may fail to detect new or unknown attacks.
There is a need for an intelligent and automated system that can continuously monitor network activity and detect threats in real time.


** Proposed Solution:**

The proposed system uses Agentic AI combined with Machine Learning to analyze incoming network traffic and classify it as normal or malicious.
The system automatically:

Monitors network traffic,

Processes and analyzes data,

Detects potential cyber threats, 

Generates alerts for suspicious activities.

This autonomous approach improves the efficiency and speed of cybersecurity monitoring.

Start the FastAPI backend server.

Launch the Streamlit dashboard.

The system begins monitoring network data.

Incoming traffic is analyzed using the ML model.

If malicious activity is detected, the system generates an alert.
